什么是订阅转换？

不同代理客户端使用不同的配置文件格式。服务商提供的订阅链接通常是通用格式， 但你的客户端可能只识别特定格式。

例如：服务商给你的是 V2Ray 格式，你想使用 Clash Verge 客户端，就可以在这里转换为 Clash YAML 格式。

本项目内置 Docker Compose 编排：一个容器运行此网页，另一个容器运行 tindy2013/subconverter 后端，适合 Ubuntu 命令行服务器部署。
